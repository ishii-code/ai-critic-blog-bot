# Score Relevance

`fetch-news` で取得した記事を批評価値でスコアリングし、最適な1件を選ぶ。

## スコアリング基準

### 高スコア（0.7以上） — 取り上げるべき

- ハイプ表現が多用された発表（「革命的」「史上最高」）
- 業界の通説に挑戦できる切り口がある
- 過去発言や別データとの矛盾を含む
- 規制・倫理に関わる事案
- 失敗・撤退・スキャンダル
- 数字で論破できる主張（コスト比較、性能比較など）

### 低スコア（0.3以下） — 避けるべき

- 単なる資金調達ニュース（数字だけ）
- 個別企業の小機能アップデート
- ハウツー・チュートリアル記事
- 明らかなPR記事・プレスリリースそのまま

## 選定ルール

1. SOUL.md の過去24時間トピックと重複しないもの
2. 過去7日のトピックとも類似しないもの
3. 同じ企業への批判が連続しないこと（SOUL.md の企業頻度確認）
4. 上記を踏まえてスコアが最も高い1件を選ぶ

## 出力形式

選んだ記事と選定理由:

```json
{
  "selected": {
    "url": "https://...",
    "title": "...",
    "summary": "...",
    "source_name": "...",
    "relevance_score": 0.85,
    "scoring_reason": "なぜこの記事を選んだか",
    "suggested_angle": "構成テンプレート番号（1-7）"
  }
}
```
